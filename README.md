# ET Mariphilia 玄理：灰烬之舟

灰烬之舟是由ET-Team制作的WIP项目，项目目标是制作一个完整的海岛魔改整合包，且继承玄理整合包系列的一贯思想。

首先，灰烬之舟和其前任：玄理及玄理2**并没有直接的从属关系及通用性**，存档并不能直接迁移且不能直接打补丁更新
